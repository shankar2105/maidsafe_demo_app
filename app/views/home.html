SAFE Network Demo's

Below are some demo's to showcase of the key network features under development.

Manage public ID

Use this demo to create Public ID to use on the SAFE Network. This ID will be used for SAFE apps e.g. Web, Drive, Messaging.

Manage websites and files

This demo is to create and manage sites hosted on the SAFE Network. Here you can choose a domain, and upload your data.

View and Manage Private Data

Here you can view and manage your private data that only you can access and see.
